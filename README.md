# pvpbot-fabric-1.21.10

A Fabric mod for Minecraft 1.21.10.

This repository contains a Fabric mod targeting Minecraft 1.21.10. The mod provides PvP-related functionality and utilities (see code for details).

## Requirements

- Java 17 or newer
- Fabric Loader for Minecraft 1.21.10
- Fabric API (if used by the mod)
- Gradle (wrapper provided: ./gradlew / gradlew.bat)

## Building

From the repository root run:

Unix/macOS:

    ./gradlew build

Windows:

    gradlew.bat build

Build artifacts (JAR) will be placed in build/libs/.

## Installing

1. Install Fabric Loader for Minecraft 1.21.10.
2. Place the built JAR (or the provided release JAR) into your Minecraft `mods/` folder.
3. Launch Minecraft with the Fabric profile.

## Usage

Consult the source code and any in-mod documentation for usage details. This README provides setup and build instructions; feature documentation may be present in the code or separate docs.
